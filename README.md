# 阿卡·迪亚 — 全屋实木定制

品牌介绍静态页面，基于 PDF 画册内容开发。

## 页面预览

双击 `arkadia.html` 即可在浏览器中查看。

## 功能

- 🏠 **品牌故事** — 源自古希腊 Arkadia 乌托邦
- 🪵 **臻选木材** — 黑胡桃、樱桃木、乌金木
- 🎨 **产品风格** — 欧美古典、现代轻奢、新中式
- ✨ **设计理念** — 大道极简，禅意肆意
- 🔍 **灯箱放大** — 点击任意图片放大查看原图，键盘 ← → ESC 控制
- 📱 **响应式** — 手机/平板自适应

## 文件结构

```
arkadia-site/
├── arkadia.html          ← 品牌介绍页面（纯 HTML + CSS，零依赖）
└── arkadia_images/       ← 74 张品牌图片（从 PDF 提取）
```

## 部署

上传到任意静态托管即可，无需后端：

- **阿里云 OSS** — 开启静态网站托管
- **GitHub Pages** — 仓库设置中启用
- **Nginx** — 指向目录即可

## 许可

内部使用
